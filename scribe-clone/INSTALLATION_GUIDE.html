Installation And Usage Guide

HachiAi Requirements Gathering Tool

This guide explains how to install the application, launch it for the first time, start and stop recordings, edit guides, annotate screenshots, export documents, and save finished workflows.

What Is Included In The Installer

  • The HachiAi desktop application
  • Electron runtime
  • Required application dependencies
  • Branding assets and installer resources
No separate Node.js, npm, Electron, or extra dependency installation is required.

Important Notes Before Use

  • The app may request Windows screen, accessibility, or input-monitoring related permissions.
  • These permissions are required for workflow capture and screenshot generation.
  • If Windows SmartScreen appears, users can select More info and then Run anyway if they trust the installer source.

Installation Steps

  1. Run the setup installer file.
  2. Complete the installation steps shown by the installer.
  3. A desktop shortcut and Start Menu shortcut will be created during installation.
  4. Launch HachiAi Requirements Gathering Tool from the desktop shortcut or Start Menu.
  5. Review and accept the privacy agreement on first launch.
  6. After acceptance, the main dashboard becomes available for creating or editing guides.

Dashboard Overview

The dashboard is the main home screen of the tool. It shows saved guides, a button to create a new recording, and options to reopen or delete existing guides.

What Users Can Do On The Dashboard

  • Click New Guide to start recording a new workflow.
  • Click an existing guide card to open it in the editor.
  • Click the delete icon on a card to remove a saved guide.
  • Review the number of steps captured in each saved guide.

How To Start And Stop Recording

  1. From the dashboard, click New Guide.
  2. The app starts capturing the workflow and shows a compact floating recording bar.
  3. Continue performing the steps you want documented.
  4. When finished, stop the recording from the floating bar or by returning to the app.
  5. After stopping, the guide opens automatically in the editor for review.

Recording Bar Controls

The floating recording bar stays visible while capture is active so the user can control recording quickly.

  • The left indicator shows that recording is active.
  • The pause or resume button controls the recording state.
  • The stop button finishes recording and opens the guide editor.

What Happens After Recording

Once recording stops, the tool converts captured actions into a structured guide with screenshots, step descriptions, application context, and editing options.

Result: A complete guide is created and opened in the editor for refinement.

Editing The Guide Title

The guide title can be edited directly inside the editor screen. The title area supports long names and wraps across lines.

  • Edit the guide title in the top title field.
  • Edit the large heading in the main editor banner if preferred.
  • Use clear descriptive names so the guide is easier to identify later.

Editing Steps

Every recorded step can be updated from the editor. Step descriptions are editable and support wrapped multi-line text.

  • Click inside a step description to rewrite or improve the instruction text.
  • Drag a step using the handle on the left to reorder the workflow.
  • Use the delete icon to remove unnecessary steps.
  • Review the screenshot preview and context shown for each step.

Annotate And Redact Screenshots

Each step screenshot can be edited to highlight important areas or hide sensitive information.

  1. Click the edit icon on a step.
  2. The screenshot opens in the annotation window.
  3. Choose highlight or redact mode.
  4. Drag on the screenshot to place the selected annotation.
  5. Apply the change and close the annotation window.

Saving The Final Guide

After editing the title, steps, and screenshots, save the guide so it stays available on the dashboard.

  • Click Save Guide in the top-right area of the editor.
  • The tool stores the guide locally and returns it to the saved guides list.

Exporting The Guide

The editor includes export options so the guide can be shared in different formats.

  • Save as HTML creates a web-style guide.
  • Save as PDF creates a printable document.
  • Save as DOCX creates a Microsoft Word document.

Tip: Finalize the guide title and step text before exporting so the document looks professional.

Recommended Everyday Workflow

  1. Launch the tool and open the dashboard.
  2. Start a new guide from the New Guide button.
  3. Complete the process you want to capture.
  4. Stop recording and review the generated guide.
  5. Edit the guide title and improve each step description.
  6. Annotate or redact screenshots where needed.
  7. Save the guide and export it in the preferred format.
